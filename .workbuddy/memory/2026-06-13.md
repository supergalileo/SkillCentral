# 2026-06-13 工作日志

## SkillCentral - 5项功能修复/新增

### 1. Agent 按钮显示真实数据
- **问题**: 卡片底部 3 个 agent 圆圈空白
- **根因**: `SkillCard::setupLargeMode()` 硬编码了3个假 agent，无名称/图标
- **修复**: 
  - `SkillCard` 新增 `setAgentList(QVector<AgentInfo>)` 方法
  - `CardWidget` 新增 `setAgentList()` 传递给所有卡片
  - 按钮显示 agent 名称首字母 + tooltip 显示完整名和路径
  - `MainWindow::onAgentToggled()` 接入 `AgentManager` 实现复制/删除文件操作

### 2. Agent 筛选栏
- **新增**: 频率筛选栏下方增加 "Agent筛选：" 动态按钮栏
- **实现**: 
  - `CardWidget` 新增 `filterByAgents()` 和 `m_filterAgentIds`
  - `matchesFilter()` 增加 agent OR 筛选逻辑（只显示已启用指定 agent 的 skill）
  - `MainWindow::loadAgents()` 动态创建筛选按钮

### 3. 设置数据持久化修复
- **问题**: 设置中修改 agent 后不保存
- **根因**: `onRequestSettings()` 传 `nullptr` 给 SettingsDialog 的 AgentManager 参数
- **修复**: 改为传递 `m_agentManager`；连接 `agentsChanged` 信号刷新列表

### 4. 允许删除默认 Agent
- **问题**: 预置 Agent (WorkBuddy/Claude Code等) 无法删除
- **修复**: 移除 `onDeleteAgent()` / `onAgentSelectionChanged()` / `onAgentContextMenu()` 中的预置保护判断

### 5. 启动扫描 + 关闭自动备份
- **启动扫描**: `setDatabaseManager()` 中检查 autoScan 设置，调用 `m_agentManager->scanAllAgents()`
- **关闭备份**: 重写 `closeEvent()`，检查 autoBackup 设置，自动导出备份到 `D:/skillLibrary/backups/auto_backup_{timestamp}.db`
